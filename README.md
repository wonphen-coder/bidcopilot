# bidcopilot
智能投标助手，快速解读招标文件核心内容

# 支持招标文件格式
1. 支持点击浏览或拖拽上传招标文件；
2. 支持DOCX、DOC、PDF、ODT、TXT格式文件，其中DOC和ODT会被转化为DOCX格式，PDF文件直接解析为TXT；
3. 推荐使用DOCX格式上传，解析准确率更高；

# 功能
1. 通过正则表达式提取招标项目关键信息，支持自定义匹配参数；提取信息包括：项目名称、项目编号、采购人、招标代理机构、采购内容、采购预算/限价、项目属性、投标保证金、合同履行期限、开标时间、投标有效期和总投标包数；
2. 支持自动提取总投标包数；
3. 支持根据目标投标包数显示对应评分标准；
4. 支持根据目标投标包数提取和导出投标文件评审条款为EXCEL文件，方便打印。
